# Memory-Card-Game
elcome to the Memory Card Game project! This interactive web-based game challenges players to match pairs of cards within a grid to win. It's a classic game that tests memory and concentration skills, suitable for players of all ages.

Features

Dynamic Grid: The game dynamically generates a grid of cards based on the chosen dimension, allowing for various levels of difficulty.
Timer and Moves Tracker: Tracks the time elapsed and the number of moves made during gameplay, providing feedback on performance.
Win Condition: Notifies the player upon successfully matching all pairs of cards with a congratulatory message.
Responsive Design: Designed to be playable on various screen sizes and devices, ensuring a seamless experience on mobile and desktop.
Technologies Used
HTML: Provides the structure and content of the game.
CSS3: Styles the game elements and handles animations.
JavaScript (ES6): Implements the game logic and interactivity.
How to Play
Start Game: Click the "Start" button to initialize the game. Alternatively, clicking on a card will start the game if it hasnâ€™t already started.
Match Cards: Click on cards to reveal their symbols. Try to remember the symbols and match pairs of identical symbols to clear them from the board.
Win: Match all pairs to win the game. A message will appear showing your total moves and the time taken to complete the game.

Game Mechanics

Game Initialization: The game begins when the "Start" button is clicked or when a card is clicked if the game hasn't started yet.

Card Flipping: Clicking on a card reveals its symbol. Players can flip up to two cards at a time.

Matching Cards:
If two flipped cards match, they remain face up and are marked as "matched".
If the cards do not match, they will flip back over after a brief pause.

Winning the Game: The game continues until all pairs are matched. A win message will be displayed with details on the number of moves and time taken.
